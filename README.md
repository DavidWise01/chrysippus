# CHRYSIPPUS & THE STOICS · IV

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **LOGIKĒ**.

CHRYSIPPUS OF SOLI (c. 279–206 BCE) & the Stoics — LOGIKĒ IV, the PIVOTAL one for computing. Aristotle reasoned about CATEGORIES; the Stoics reasoned about whole STATEMENTS joined by CONNECTIVES — and, or, not, IF…THEN — the first PROPOSITIONAL logic. ⚑ THE BRIDGE TO THE GATE: a proposition is true/false = the BIT; the connectives = the OPERATORS; Philo of Megara's conditional 'if P then Q' is FALSE in exactly one case (P true, Q false) = the MATERIAL CONDITIONAL = the modern TRUTH TABLE, written ~300 BCE (verified 1011). LIVE: (1) toggle P/Q → the connectives compute (¬, ∧, ∨, ⊻, →) + the full 4-row truth table with the current row lit — every box a logic gate 2,200 years before silicon; (2) the FIVE INDEMONSTRABLES (modus ponens 'if P then Q; P; ∴ Q'; modus tollens; conj-denial; excl-disjunction; disjunctive syllogism) — the first natural-deduction rules — lighting up when their premises hold. CONFIDENCE-TIERED: STRONG (they built propositional logic — connectives, truth-functional conditional, inference rules — the genuine ancestor of Boolean logic & the gate, per Łukasiewicz/Mates/Bobzien) / RECONSTRUCTED (Chrysippus wrote 705 books, NONE survive whole — rebuilt from hostile critics Sextus Empiricus & Diogenes Laertius; the precise truth-functional reading is partly a 20th-c. reconstruction) / VINDICATED LATE (dismissed as inferior to Aristotle for ~2000 years, until propositional logic became the spine of computing). ⚑ A FIRST THAT WAS FORGOTTEN — and turned out the more important one; this is the logic your processor actually runs on.
